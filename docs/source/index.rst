scikit-multilearn's documentation!
=============================================

A scikit-learn based module for multi-label classification. 

The first release consists only of the python interface to the magnificent Jesse Read's library - `MEKA <http://meka.sourceforge.net/>`_  which gives access to all the industry standard classifiers from `WEKA <http://www.cs.waikato.ac.nz/~ml/>`_ and `MULAN <http://mulan.sourceforge.net/>`_.
